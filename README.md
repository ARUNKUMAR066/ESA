# Contains backend files for [Exam Seat Arrangement System](https://github.com/Sudhi27Krishna/Exam-Seat-Arrangement-System)


1. Clone the repository
2. cd into the file
3. Type the command 
 
    ```
    mkdir updatedExcels
    ```
 4. To run the server type the command and press <kbd>Enter</kbd>
  
    ```
    npm run dev
    ```
